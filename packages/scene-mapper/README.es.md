# @soundweave/scene-mapper

Activa la lógica de mapeo para la resolución de escenas en Soundweave.

## Funcionalidades

- Evaluación de condiciones de activación.
- Resolución de prioridades de enlace.
- Selección de escenas a partir del estado en tiempo de ejecución.

## Estado

Paquete provisional. La lógica de resolución de escenas actualmente reside en los tipos de `@soundweave/schema` y en la tienda de Studio. Este paquete se ampliará a medida que aumente la superficie de integración en tiempo de ejecución (Fase 20+).

## Funcionalidades que no posee

- Composición de escenas o gestión de capas.
- Ejecución de transiciones.
- Reproducción de audio.
