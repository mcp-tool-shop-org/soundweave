# @soundweave/scene-mapper

Dispara a lógica de mapeamento para a resolução de cenas no Soundweave.

## O que este pacote gerencia:

- Avaliação das condições de disparo.
- Resolução da prioridade de vinculação.
- Seleção de cenas a partir do estado em tempo de execução.

## Status

Pacote temporário. A lógica de resolução de cenas atualmente reside nos tipos e na loja do `@soundweave/schema`. Este pacote será expandido quando a superfície de integração em tempo de execução crescer (Fase 20+).

## O que este pacote não gerencia:

- Composição de cenas ou gerenciamento de camadas.
- Execução de transições.
- Reprodução de áudio.
