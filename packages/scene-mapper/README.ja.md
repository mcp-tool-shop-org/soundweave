# @soundweave/scene-mapper

Soundweaveのシーン解決のためのマッピングロジックをトリガーします。

## このパッケージが担当する機能

- トリガー条件の評価
- バインディングの優先順位の解決
- 実行時の状態からのシーンの選択

## ステータス

プレースホルダーパッケージです。現在のシーン解決ロジックは、`@soundweave/schema`の型定義とStudioストアに存在します。このパッケージは、ランタイム統合機能が拡張される際に（フェーズ20以降）、拡充される予定です。

## このパッケージが担当しない機能

- シーンの構成やレイヤー管理
- トランジションの実行
- オーディオ再生
