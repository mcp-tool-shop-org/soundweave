# @soundweave/scene-mapper

Trigger mapping logic for Soundweave scene resolution.

## What It Owns

- Trigger condition evaluation
- Binding priority resolution
- Scene selection from runtime state

## Status

Placeholder package. Scene resolution logic currently lives in `@soundweave/schema` types and Studio store. This package will be expanded when the runtime integration surface grows (Phase 20+).

## What It Does Not Own

- Scene composition or layer management
- Transition execution
- Audio playback
