# @soundweave/runtime-pack

Soundweave サウンドトラックパックの実行時エクスポートとインポート機能。

## 機能概要

- 制作用パックからの実行時用パックの不要な部分の削除
- 実行時用パックのシリアライズ（データ形式への変換）
- インポート/エクスポートの相互検証

## 状態

プレースホルダーパッケージです。フェーズ20（公開、パッケージング、および実行時統合）において、エクスポートプロファイル、アセットのバンドリング、およびエンジンアダプターのマニフェストを含む、実行時エクスポートの機能が拡張されます。

## 含まれない機能

- 制作データモデル（`@soundweave/schema` を参照）
- オーディオファイルのパッケージング
- エンジン固有の再生SDK
