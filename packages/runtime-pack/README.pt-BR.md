# @soundweave/runtime-pack

Exportação e importação em tempo de execução para pacotes de trilha sonora do Soundweave.

## O que este pacote inclui:

- Remoção de elementos específicos da criação para gerar pacotes de tempo de execução.
- Serialização de pacotes de tempo de execução.
- Verificação de importação/exportação em ciclo fechado.

## Status

Pacote temporário. A lógica de exportação em tempo de execução será expandida na Fase 20 (Publicação, Empacotamento e Integração em Tempo de Execução) para incluir perfis de exportação, agrupamento de ativos e arquivos de manifesto do motor.

## O que este pacote não inclui:

- Modelo de dados de criação (veja `@soundweave/schema`).
- Empacotamento de arquivos de áudio.
- SDKs de reprodução específicos para cada motor.
