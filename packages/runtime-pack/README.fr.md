# @soundweave/runtime-pack

Exportation et importation en temps réel pour les packs de bandes sonores Soundweave.

## Ce que ce module contient

- Suppression des éléments spécifiques à la création pour convertir un pack de création en un pack en temps réel.
- Sérialisation des packs en temps réel.
- Vérification de l'importation/exportation en aller-retour.

## Statut

Ce package est un espace réservé. La logique d'exportation en temps réel sera étendue dans la phase 20 (Publication, empaquetage et intégration en temps réel) pour inclure les profils d'exportation, le regroupement des ressources et les manifestes d'adaptateurs de moteur.

## Ce que ce module ne contient pas

- Modèle de données de création (voir `@soundweave/schema`).
- Emballage des fichiers audio.
- SDK de lecture spécifiques à chaque moteur.
