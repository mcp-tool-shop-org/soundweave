# @soundweave/runtime-pack

साउंडवीव साउंडट्रैक पैकों के लिए रनटाइम एक्सपोर्ट और इम्पोर्ट।

## इसमें क्या शामिल है

- ऑथरिंग पैक से रनटाइम पैक बनाना (अनावश्यक तत्वों को हटाना)।
- रनटाइम पैक का सीरियललाइज़ेशन।
- इम्पोर्ट/एक्सपोर्ट की प्रक्रिया की जांच।

## स्थिति

यह एक अस्थायी पैकेज है। दूसरे चरण (पब्लिशिंग, पैकेजिंग और रनटाइम इंटीग्रेशन) में, रनटाइम एक्सपोर्ट लॉजिक को बढ़ाया जाएगा, जिसमें एक्सपोर्ट प्रोफाइल, एसेट बंडलिंग और इंजन एडाप्टर मैनिफेस्ट शामिल होंगे।

## इसमें क्या शामिल नहीं है

- ऑथरिंग डेटा मॉडल ( `@soundweave/schema` देखें)।
- ऑडियो फ़ाइल पैकेजिंग।
- इंजन-विशिष्ट प्लेबैक SDKs।
