# @soundweave/review

Summaries, audits, and preview helpers for Soundweave soundtrack packs.

## What It Owns

- Pack summary generation
- Audit result formatting
- Preview helper utilities

## Status

Placeholder package. Review and summary logic currently lives in `@soundweave/asset-index` (integrity auditing) and Studio screens (Overview). This package will grow to include richer reporting, cue coverage analysis, and review annotations (Phase 26–27).

## What It Does Not Own

- Pack integrity indexing (see `@soundweave/asset-index`)
- Schema validation (see `@soundweave/schema`)
- UI rendering
