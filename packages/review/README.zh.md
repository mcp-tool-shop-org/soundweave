# @soundweave/review

用于 Soundweave 音频素材包的摘要、审计和预览辅助工具。

## 功能

- 音频素材包摘要生成
- 审计结果格式化
- 预览辅助工具

## 状态

这是一个占位符包。当前的摘要和审查逻辑位于 `@soundweave/asset-index`（完整性审计）和 Studio 界面（概览）中。该包将不断扩展，以包含更丰富的报告、音效覆盖分析以及审查注释（第二阶段 26-27）。

## 不包含的功能

- 音频素材包完整性索引（请参见 `@soundweave/asset-index`）
- 模式验证（请参见 `@soundweave/schema`）
- 用户界面渲染
