# @soundweave/ui

Componentes de interface de usuário (UI) compartilhados para aplicações Soundweave.

## O que este pacote inclui:

- Elementos básicos de UI reutilizáveis, compartilhados entre o Studio e a documentação.
- Componentes do sistema de design (botões, campos de entrada, abas, selos, etc.).

## Status:

Pacote temporário. Os componentes de UI atualmente estão integrados diretamente nos arquivos de tela do Studio. Este pacote será extraído e unificado durante a fase de refinamento visual (Fase 22).

## O que este pacote não inclui:

- Gerenciamento do estado da aplicação (ver "Studio store").
- Layout e roteamento da tela.
- Lógica de negócios.
