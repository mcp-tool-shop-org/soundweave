# @soundweave/ui

साउंडवीव अनुप्रयोगों के लिए साझा यूआई घटक।

## इसमें क्या शामिल है

- स्टूडियो और डॉक्स के बीच साझा किए जाने वाले पुन: प्रयोज्य यूआई तत्व।
- डिज़ाइन सिस्टम घटक (बटन, इनपुट, टैब, बैज, आदि)।

## स्थिति

यह एक अस्थायी पैकेज है। वर्तमान में, यूआई घटक स्टूडियो स्क्रीन फ़ाइलों में सीधे मौजूद हैं। इस पैकेज को दृश्य सुधार चरण (चरण 22) के दौरान निकाला और एकीकृत किया जाएगा।

## इसमें क्या शामिल नहीं है

- एप्लिकेशन स्टेट प्रबंधन (स्टूडियो स्टोर देखें)।
- स्क्रीन लेआउट और रूटिंग।
- व्यावसायिक तर्क।
