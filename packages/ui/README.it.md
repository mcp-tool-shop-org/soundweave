# @soundweave/ui

Componenti dell'interfaccia utente (UI) condivisi per le applicazioni Soundweave.

## Cosa include

- Elementi primitivi dell'interfaccia utente riutilizzabili, condivisi tra Studio e la documentazione.
- Componenti del sistema di progettazione (pulsanti, campi di input, schede, badge, ecc.).

## Stato

Pacchetto di esempio. Attualmente, i componenti dell'interfaccia utente sono integrati direttamente nei file di schermata di Studio. Questo pacchetto verrà estratto e unificato durante la fase di rifinitura visiva (Fase 22).

## Cosa non include

- Gestione dello stato dell'applicazione (vedere lo store di Studio).
- Layout e routing delle schermate.
- Logica di business.
