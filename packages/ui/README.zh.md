# @soundweave/ui

Soundweave应用程序的共享UI组件。

## 包含内容

- 可重用的UI基础组件，在Studio和文档之间共享。
- 设计系统组件（按钮、输入框、标签、徽章等）。

## 状态

占位符包。目前，UI组件直接嵌入在Studio的屏幕文件中。该包将在视觉优化阶段（第二阶段）被提取和统一。

## 不包含的内容

- 应用程序状态管理（请参见Studio store）。
- 屏幕布局和路由。
- 业务逻辑。
