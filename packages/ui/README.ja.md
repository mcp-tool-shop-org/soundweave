# @soundweave/ui

Soundweaveアプリケーションで使用される共通UIコンポーネント。

## このパッケージに含まれるもの

- StudioとDocsで共有される再利用可能なUIの基本要素
- デザインシステムコンポーネント（ボタン、入力フィールド、タブ、バッジなど）

## 状態

プレースホルダーパッケージです。現在のUIコンポーネントは、Studioの画面ファイル内に直接記述されています。このパッケージは、デザインの調整段階（フェーズ22）で抽出され、統合されます。

## このパッケージに含まれないもの

- アプリケーションの状態管理（Studioのストアを参照）
- 画面のレイアウトとルーティング
- ビジネスロジック
