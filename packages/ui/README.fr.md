# @soundweave/ui

Composants d'interface utilisateur (UI) partagés pour les applications Soundweave.

## Ce que ce module contient

- Éléments primitifs de l'interface utilisateur réutilisables, partagés entre Studio et la documentation.
- Composants du système de conception (boutons, champs de saisie, onglets, badges, etc.).

## Statut

Paquet temporaire. Les composants de l'interface utilisateur sont actuellement intégrés directement dans les fichiers d'écran de Studio. Ce paquet sera extrait et unifié lors de la phase de perfectionnement visuel (Phase 22).

## Ce que ce module ne contient pas

- Gestion de l'état de l'application (voir le magasin de Studio).
- Disposition et routage des écrans.
- Logique métier.
