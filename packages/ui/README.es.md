# @soundweave/ui

Componentes de interfaz de usuario compartidos para aplicaciones de Soundweave.

## Qué incluye

- Elementos básicos de la interfaz de usuario reutilizables, compartidos entre Studio y Docs.
- Componentes del sistema de diseño (botones, campos de entrada, pestañas, insignias, etc.).

## Estado

Paquete provisional. Los componentes de la interfaz de usuario actualmente se encuentran integrados en los archivos de pantalla de Studio. Este paquete se extraerá y unificará durante la fase de mejora visual (Fase 22).

## Qué no incluye

- Gestión del estado de la aplicación (ver almacén de Studio).
- Diseño y enrutamiento de la pantalla.
- Lógica de negocio.
