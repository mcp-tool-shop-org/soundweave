# @soundweave/docs

Soundweave 的文档网站，使用 Astro 构建。

## 页面

- **首页** — 概述和导航
- **入门** — 设置和初步步骤
- **核心概念** — 实体、模式和数据模型
- **工作流程** — 内容创作流程和创意模式
- **示例** — 示例包和用法

## 开发

```bash
pnpm --filter @soundweave/docs dev    # Start dev server
pnpm --filter @soundweave/docs build  # Build static site
```
