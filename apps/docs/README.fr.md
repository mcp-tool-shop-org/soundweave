# @soundweave/docs

Site de documentation pour Soundweave, créé avec Astro.

## Pages

- **Accueil** — aperçu et navigation
- **Premiers pas** — installation et premières étapes
- **Concepts clés** — entités, schémas et modèle de données
- **Flux de travail** — création de flux de travail et de modèles créatifs
- **Exemples** — ensembles d'exemples et utilisation

## Développement

```bash
pnpm --filter @soundweave/docs dev    # Start dev server
pnpm --filter @soundweave/docs build  # Build static site
```
