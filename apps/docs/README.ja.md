# @soundweave/docs

Soundweaveのドキュメントサイト。Astroで構築されています。

## ページ

- **ホーム** - 概要とナビゲーション
- **はじめに** - セットアップと最初のステップ
- **基本概念** - エンティティ、スキーマ、データモデル
- **ワークフロー** - 制作ワークフローとクリエイティブなパターン
- **例** - サンプルセットと使用方法

## 開発

```bash
pnpm --filter @soundweave/docs dev    # Start dev server
pnpm --filter @soundweave/docs build  # Build static site
```
