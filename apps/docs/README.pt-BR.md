# @soundweave/docs

Site de documentação do Soundweave, construído com Astro.

## Páginas

- **Início** — visão geral e navegação
- **Primeiros Passos** — configuração e primeiros passos
- **Conceitos Fundamentais** — entidades, esquemas e modelo de dados
- **Fluxo de Trabalho** — fluxos de trabalho e padrões criativos
- **Exemplos** — pacotes de exemplos e uso

## Desenvolvimento

```bash
pnpm --filter @soundweave/docs dev    # Start dev server
pnpm --filter @soundweave/docs build  # Build static site
```
