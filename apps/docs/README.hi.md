# @soundweave/docs

साउंडवीव के लिए दस्तावेज़ वेबसाइट, जो एस्ट्रो के साथ बनाई गई है।

## पृष्ठ

- **होम** — अवलोकन और नेविगेशन
- **शुरुआत कैसे करें** — सेटअप और शुरुआती कदम
- **मुख्य अवधारणाएं** — एंटिटीज, स्कीमा और डेटा मॉडल
- **कार्यप्रणाली** — रचना प्रक्रियाएं और रचनात्मक पैटर्न
- **उदाहरण** — उदाहरण संग्रह और उपयोग

## विकास

```bash
pnpm --filter @soundweave/docs dev    # Start dev server
pnpm --filter @soundweave/docs build  # Build static site
```
