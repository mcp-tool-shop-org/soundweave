यह एक अनुकूलनीय साउंडट्रैक स्टूडियो है, जिसका उपयोग इंटरैक्टिव गेम संगीत बनाने, व्यवस्थित करने, स्कोर करने और निर्यात करने के लिए किया जाता है।

## यह क्या है

साउंडवीव एक ऐसा वर्कस्टेशन है जो रचना पर केंद्रित है और अनुकूलन के लिए तैयार है। यह संरचित संगीत रचना (क्लिप, क्यू, सीन, लेयर, ऑटोमेशन) को अनुकूलनीय लॉजिक के साथ जोड़ता है जो गेम की स्थिति के अनुसार बदलता है। इसका परिणाम: ऐसा गेम संगीत जो जानबूझकर बनाया गया लगता है, न कि उत्पन्न किया गया।

## यह क्या नहीं है

यह एक डिजिटल ऑडियो वर्कस्टेशन (DAW) नहीं है। यह एक साधारण सीक्वेंसिंग टूल नहीं है। यह एक कृत्रिम बुद्धिमत्ता (AI) संगीत जनरेटर नहीं है। यह एक ऐसा डेटाबेस नहीं है जिसमें ध्वनि जुड़ी हुई है। साउंडवीव एक गंभीर रचनात्मक उपकरण है जिसका उपयोग अनुकूलनीय गेम स्कोर बनाने के लिए किया जाता है।

## यह क्या कर सकता है

- **रचना:** नोट्स, वाद्य यंत्र, स्केल, कॉर्ड, मोटिफ रूपांतरण, तीव्रता भिन्नताएं वाली क्लिप।
- **सिंथेसाइज़:** मल्टी-ऑसिलेटर सिंथ आवाजें (16 प्रीसेट), यूनिसन/सुपरसॉ, एलएफओ मॉड्यूलेशन (फ़िल्टर, एम्प्लिट्यूड, पिच)।
- **सैंपल वाद्य यंत्र:** पियानो, स्ट्रिंग, गिटार टेम्पलेट (सैंपलवॉइस के माध्यम से); इम्पोर्ट, ट्रिम, स्लाइस, किट बिल्डर।
- **व्यवस्था:** लेयर्ड स्टेम, सेक्शन भूमिकाएं, तीव्रता वक्र वाले सीन; 10 ड्रम पैटर्न प्रीसेट।
- **मिक्स और इफेक्ट:** 8 इफेक्ट प्रकार (EQ, डिले, रिवर्ब, कंप्रेसर, कोरस, डिस्टॉर्शन, फेजर, लिमिटर); प्रति स्टेम 4 इंसर्ट FX स्लॉट।
- **दुनिया को स्कोर करें:** मोटिफ परिवार, स्कोर प्रोफाइल, क्यू परिवार, वर्ल्ड मैप एंट्री, डेरिवेशन।
- **ऑटोमेट:** लेन, मैक्रो, एनवेलप, लाइव कैप्चर और मर्ज।
- **पुनर्प्राप्ति और पुन: उपयोग:** टेम्पलेट, स्नैपशॉट, ब्रांच, पसंदीदा, संग्रह, तुलना।
- **MIDI:** स्टैंडर्ड MIDI फ़ाइलें इम्पोर्ट/एक्सपोर्ट करें।
- **अनुकूलन लॉजिक:** ट्रिगर बाइंडिंग, ट्रांज़िशन, नियतात्मक सीन रिज़ॉल्यूशन।
- **प्रदर्शन:** रीयल-टाइम क्लिप प्रीव्यू, ऑडिशन के लिए क्लिक, ऑडियोकॉन्टेक्स्ट-शेड्यूल्ड क्लिक के साथ मेट्रोनोम।
- **सत्यापन:** स्कीमा सत्यापन, इंटीग्रिटी ऑडिटिंग, क्रॉस-रेफरेंस चेक।
- **निर्यात:** 44.1/48/96kHz पर 24/32-बिट WAV; गेम इंजन के उपयोग के लिए रनटाइम पैक।
- **लेखक:** अनडू/रीडू (50 तक, Ctrl+Z), प्रोजेक्ट सेव/लोड (ऑटोसेव के साथ), कीबोर्ड शॉर्टकट (स्पेस=प्ले, ?=मदद), ग्लोबल बीपीएम और टाइम सिग्नेचर।
- **विश्वसनीयता:** त्रुटि सीमा, त्रुटि से उबरने की क्षमता, सैंपल-एक्कुरेट टाइमिंग के लिए ऑडियोकॉन्टेक्स्ट लुकअहेड शेड्यूलिंग।

## इंस्टॉल करें

```bash
npm install @soundweave/schema @soundweave/clip-engine @soundweave/runtime-pack
```

सभी पैकेज `@soundweave` स्कोप के तहत npm पर प्रकाशित किए गए हैं।

## शुरुआत (मोनोरेपो)

```bash
pnpm install
pnpm build
pnpm test       # 1,002 tests across all packages
pnpm dev        # Start Studio dev server
```

**आवश्यकताएं:** Node.js >= 22, pnpm >= 10

## परीक्षण

सभी 16 पैकेजों में यूनिट टेस्ट शामिल हैं जो स्कीमा सत्यापन, डेटा अखंडता जांच, नमूना संचालन, वर्ल्ड स्कोरिंग, ऑटोमेशन, लाइब्रेरी प्रबंधन, प्लेबैक, संश्लेषण, प्रभाव, MIDI और स्टूडियो एकीकरण को कवर करते हैं। सभी पैकेजों में कुल 1,002 परीक्षण हैं।

सब कुछ चलाने के लिए: `pnpm test`

## सुरक्षा और विश्वसनीयता

साउंडवीव पूरी तरह से **ब्राउज़र में** चलता है। कोई सर्वर नहीं, कोई क्लाउड सिंक नहीं, कोई टेलीमेट्री नहीं।

- **डेटा जिस पर काम किया जाता है:** उपयोगकर्ता द्वारा बनाए गए साउंडट्रैक पैकेज फ़ाइलें (JSON), ऑडियो एसेट संदर्भ, ब्राउज़र लोकल स्टोरेज।
- **डेटा जिस पर काम नहीं किया जाता है:** कोई सर्वर-साइड स्टोरेज नहीं, ब्राउज़र सैंडबॉक्स से परे कोई फ़ाइल सिस्टम एक्सेस नहीं।
- **नेटवर्क:** शून्य नेटवर्क आउटगोइंग - सभी रचना और प्लेबैक क्लाइंट-साइड पर होता है।
- **गुप्त जानकारी:** यह क्रेडेंशियल्स को नहीं पढ़ता, संग्रहीत नहीं करता या प्रसारित नहीं करता।
- **टेलीमेट्री:** कोई भी डेटा एकत्र या भेजा नहीं जाता।
- **अनुमतियाँ:** केवल मानक ब्राउज़र एपीआई (वेब ऑडियो एपीआई) का उपयोग किया जाता है।

भेद्यता रिपोर्टिंग के लिए [SECURITY.md](SECURITY.md) देखें।
