एक ऐसा सॉफ्टवेयर स्टूडियो जो इंटरैक्टिव गेम के संगीत को बनाने, व्यवस्थित करने, रिकॉर्ड करने और निर्यात करने के लिए उपयोग किया जाता है, और जो उपयोगकर्ता की आवश्यकताओं के अनुसार अनुकूलित किया जा सकता है।

## यह क्या है।

साउंडवीव एक ऐसा सॉफ्टवेयर है जो संगीत रचना को प्राथमिकता देता है और अनुकूलन क्षमता को ध्यान में रखता है। यह संरचित संगीत रचना की सुविधाएँ—जैसे कि क्लिप, क्यू, सीन, लेयर और ऑटोमेशन—को एक ऐसे लॉजिक के साथ जोड़ता है जो गेम के चलने के दौरान उसकी स्थिति के अनुसार प्रतिक्रिया करता है। इसका परिणाम यह होता है कि गेम का संगीत ऐसा लगता है कि वह जानबूझकर बनाया गया है, न कि किसी स्वचालित प्रक्रिया द्वारा उत्पन्न किया गया है।

## यह क्या नहीं है।

एक डिजिटल ऑडियो वर्कस्टेशन (DAW)। एक खिलौना सीक्वेंसर। एक कृत्रिम बुद्धिमत्ता (AI) आधारित संगीत जनरेटर। एक ऐसा डेटाबेस जो ध्वनि से जुड़ा है और जिसका उपयोग दुनिया बनाने में किया जा सकता है। साउंडवीव एक गंभीर रचनात्मक उपकरण है, जिसका उपयोग अनुकूलनीय गेम स्कोर बनाने के लिए किया जाता है।

## यह क्या कर सकता है।

- **रचना (Compose)** — नोट्स, वाद्य यंत्र, स्केल, कॉर्ड, मोटिफ परिवर्तन, तीव्रता भिन्नताएं वाले क्लिप।
- **व्यवस्था (Arrange)** — परतों वाले स्टम्स, खंडों की भूमिकाएं, तीव्रता वक्र वाले दृश्य।
- **एक दुनिया का निर्माण (Score a world)** — मोटिफ परिवार, स्कोर प्रोफाइल, क्यू परिवार, विश्व मानचित्र प्रविष्टियां, व्युत्पत्ति।
- **स्वचालित करें (Automate)** — लेन, मैक्रो, एनवेलप, लाइव कैप्चर और विलय।
- **पुनर्प्राप्त करें और पुनः उपयोग करें (Recall and reuse)** — टेम्पलेट, स्नैपशॉट, शाखाएं, पसंदीदा, संग्रह, तुलना।
- **सैंपल वर्कफ़्लो (Sample workflow)** — आयात, ट्रिम, स्लाइस, किट बिल्डर, सैंपल वाद्य यंत्र।
- **अनुकूलनशील तर्क (Adaptive logic)** — ट्रिगर बाइंडिंग, संक्रमण, नियतात्मक दृश्य समाधान।
- **सत्यापित करें (Validate)** — स्कीमा सत्यापन, अखंडता ऑडिट, क्रॉस-रेफरेंस जांच।
- **निर्यात करें (Export)** — गेम इंजन द्वारा उपयोग किए जाने वाले रनटाइम पैकेज।

## शुरुआत कैसे करें।

```bash
pnpm install
pnpm build
pnpm test       # 299+ tests across all packages
pnpm dev        # Start Studio dev server
```

**आवश्यकताएं:** Node.js >= 22, pnpm >= 10।

## परीक्षण।

सभी पैकेजों में इकाई परीक्षण शामिल हैं जो स्कीमा सत्यापन, डेटा अखंडता की जांच, नमूना संचालन, विश्व स्कोरिंग, स्वचालन, लाइब्रेरी प्रबंधन और स्टूडियो एकीकरण जैसे पहलुओं को कवर करते हैं।

सभी परीक्षण चलाने के लिए: `pnpm test` का उपयोग करें।

## निर्देशिका/मार्गदर्शिका।

यह [मैन्युअल](handbook/) एक विस्तृत परिचालन मार्गदर्शिका है जिसमें उत्पाद की अवधारणा, आर्किटेक्चर, डेटा मॉडल, स्टूडियो का उपयोग, रचनात्मक कार्यप्रणाली और इंजीनियरिंग पद्धतियों (40 अध्याय) को शामिल किया गया है।

## सुरक्षा और विश्वास।

साउंडवीव पूरी तरह से ब्राउज़र में ही चलता है। इसमें किसी सर्वर की आवश्यकता नहीं होती, न ही क्लाउड सिंक की, और न ही किसी प्रकार के डेटा संग्रह (टेलीमेट्री) की।

- **जिन डेटा तक पहुंचा गया:** उपयोगकर्ता द्वारा बनाए गए साउंडट्रैक पैकेज फ़ाइलें (JSON), ऑडियो संसाधनों के संदर्भ, ब्राउज़र का स्थानीय स्टोरेज।
- **जिन डेटा तक नहीं पहुंचा गया:** कोई भी सर्वर-साइड स्टोरेज, ब्राउज़र सैंडबॉक्स से बाहर फ़ाइल सिस्टम तक कोई पहुंच नहीं।
- **नेटवर्क:** कोई भी नेटवर्क से डेटा बाहर नहीं भेजा जाता - सभी सामग्री निर्माण और प्लेबैक क्लाइंट-साइड पर होता है।
- **सुरक्षा:** यह किसी भी प्रकार के क्रेडेंशियल को नहीं पढ़ता, संग्रहीत नहीं करता, और न ही प्रसारित करता है।
- **डेटा संग्रह:** कोई भी डेटा एकत्र या भेजा नहीं जाता।
- **अनुमतियाँ:** केवल मानक ब्राउज़र एपीआई (वेब ऑडियो एपीआई) का उपयोग किया जाता है।

सुरक्षा संबंधी जानकारी के लिए, [SECURITY.md](SECURITY.md) देखें।
