# Clips Screen

The Clips screen is the composition workspace.

## Features

- Create and edit clips with note sequences
- Apply transforms: transpose, invert, reverse, octave shift, rhythm scale
- Generate variations: rhythmic, melodic, thin, densify, accents, ghost hits
- Derive intensity variants: low, mid, high, tension, brighten
- Use chord tools: diatonic chords, chord palette, progressions
- Snap notes to scale
- Assign clips to scenes via clip layers

## Composition Workflow

1. Create a clip and enter notes
2. Set the scale/key context
3. Apply transforms and variations to explore alternatives
4. Derive intensity variants for adaptive layering
5. Assign the clip (and its variants) to scenes
