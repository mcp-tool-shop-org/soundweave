# Automation Screen

The Automation screen provides curve editing and performance capture.

## Features

- Create and edit automation lanes targeting any parameter
- Visualize interpolated values over time
- Define macros (intensity, tension, energy) and map them to parameters
- Create section envelopes for cue-level dynamics
- Record live captures during preview
- Thin, apply, and merge captured data into lanes

## Workflow

1. Create lanes targeting parameters (stem gain, effect values, etc.)
2. Draw or record automation curves
3. Set up macros for high-level creative control
4. Use section envelopes for structural dynamics
5. Capture live parameter changes during preview playback
6. Thin captured data and merge into lanes
