# Score Map Screen

The Score Map screen connects music to game world structure.

## Features

- Create and manage motif families (variants, scene links)
- Define score profiles (tempo, intensity, palette, key/scale per context)
- Build cue families (grouped scenes by narrative role, motif links)
- Create score map entries (region/faction/encounter → profile + families)
- Track derivation records (source → transform → target)

## Workflow

1. Identify the motifs (thematic ideas) in your score
2. Group them into motif families
3. Define score profiles for each region, faction, or encounter type
4. Organize scenes into cue families by narrative role
5. Create score map entries linking game contexts to profiles and families
6. Use derivation to evolve material while preserving lineage
