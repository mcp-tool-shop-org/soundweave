# Assets Screen

The Assets screen manages the audio asset registry.

## Features

- Add, edit, and remove audio assets
- Set kind (loop, oneshot, stinger, ambient)
- Configure BPM, key, tags, duration
- Set trim points and loop regions
- View import metadata (source type, original filename)

## Workflow

1. Import audio files (or create assets manually)
2. Set kind and musical metadata
3. Optionally configure trim and loop points
4. Assets become available for stem creation

Assets are the raw material. Everything else in Soundweave builds on top of them.
