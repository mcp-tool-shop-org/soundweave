# Sample Lab

The Sample Lab screen provides the import-to-instrument pipeline.

## Features

- **Trim** — set non-destructive start/end boundaries
- **Loop** — define loop regions within assets
- **Slice** — divide assets into even or onset-based slices
- **Kit Builder** — map slices to MIDI pitches, creating drum-style kits
- **Instrument Builder** — wrap assets as pitched instruments with root note and range

## Workflow

1. Select an asset
2. Trim to isolate the useful region
3. Slice into individual hits or notes
4. Build a kit (percussion) or instrument (pitched) from the slices
5. Use the kit/instrument in clips and scenes

The Sample Lab makes it possible to turn any audio recording into a playable source without leaving Soundweave.
