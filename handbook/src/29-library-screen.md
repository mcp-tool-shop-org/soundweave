# Library Screen

The Library screen manages creative recall and reuse.

## Features

- **Templates** — save reusable patterns, instantiate new entities from templates
- **Snapshots** — freeze entity state, browse history, restore previous versions
- **Branches** — fork from snapshots, trace lineage, explore variations
- **Favorites** — bookmark entities for quick access
- **Collections** — organize favorites into named groups
- **Compare** — side-by-side entity diff, structural equality check, version promotion

## Workflow

1. Save a template from any entity you want to reuse
2. Take snapshots before making experimental changes
3. Branch from a snapshot to try variations without losing the original
4. Favorite entities you return to frequently
5. Organize favorites into collections for project phases or themes
6. Compare versions to decide which to keep
